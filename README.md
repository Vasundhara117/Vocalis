#  Vocalis — Voice-Based Speech Therapy Platform

A full-stack web app that helps users improve pronunciation using real-time speech recognition and gamified learning.

---

##  Problem Statement

Many learners struggle with pronunciation due to lack of:

* Real-time feedback
* Consistent speaking practice
* Engaging learning systems

---

##  Features

* Real-time speech recognition using Deepgram API
*  Gamified levels (alphabets → words → sentences)
*  Personalized practice based on weak words
*  Leaderboard, streaks, and achievements

---

##  Tech Stack

Frontend: React
Backend: Node.js, Express
Database: MongoDB
Auth: JWT, bcrypt
API: Deepgram

---

##  Setup & Run

```bash
git clone https://github.com/Vasundhara117/Vocalis.git
cd Vocalis
npm install
npm start
```

Create `.env` file:

```env
DEEPGRAM_API_KEY=your_api_key
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
```
